# Trợ lí AI hỗ trợ học sinh học Tin học 10

## Chức năng
- Giao diện sáng, dễ dùng
- Chế độ Học bài
- Chế độ Câu trắc nghiệm
- Chế độ Đề kiểm tra
- Câu hỏi gợi ý bám sát Tin học 10 sách Kết nối tri thức
- Xóa hội thoại, xuất hội thoại TXT

## Chạy trên máy
```bash
npm install
npm start
```

Mở trình duyệt tại:
```bash
http://localhost:10000
```

## Triển khai Render
- Build Command: `npm install`
- Start Command: `npm start`
- Environment Variable: `GEMINI_API_KEY`

## Lưu ý
- GitHub Pages không chạy được backend Node.js.
- Muốn chatbot trả lời bằng AI thật, cần deploy backend lên Render hoặc nền tảng tương tự.
